Grundprinzip

Der Physical Semantic Object Annotator (PSOA) baut auf dem Semantic Object Data Model (SOM) auf. Als Grundelement dient das semantische Objekt (SO). Dies wird durch eine Open Annotation Annotation (inkl. Metadaten) mittels Target und Body beschrieben (A). Zur Identifzierung des semantischen Objekts ist mindestens eine textliche Information nötig, das präferierte Label (PL). ein semantisches Objekt ohne prefLabel wird im PSOA nicht angezeigt. Wird eine Annotation mit dem prefLabel gelöscht, ist das semantische Objekt als ebenfalls gelöscht anzusehen.

Neues Semantic Object erstellen

Existiert kein semantisches Objekt, kann mit PSOA und dem Semantic Object Service ein neues im Web adessierbares semantisches Objekt erzuegt werden. PSOA ermöglicht die Annotation dieses Objekts in textlicher Weise durch das präferierte Label (PL), ein optionales alternatives Label (AL) und eine optionale Beschreibung (SN). Des Weiteren sind optional die Klassifizierung über ein Label (L), das Hinzufügen weiterer Informationen über Expert Data (ED), sowie weitere Beschreibungen mit Literalen ("...") bzw. HTTP Resourcen (http://...) möglich.

Semantic Object annotieren

Besteht bereits ein semantisches Objekt, kann dies erneut vom gleichen oder einem anderen Annotator annotiert werden. Hierbei dient das Target als verbindendes Element.

Feature annotieren

Der Physical Semantic Object Annotator (PSOA) ist primär für die Annotation von bereits bestehenden Feature Geometrien des Projekts IBR (Inschriften im Bezugssystem des Raumes) konzipiert. Hierbei müssen die Geometrien annotiert und zuvor ein semantisches Objekt erstellt werden. Hierzu wird ebenfalls der Semantic Object Service genutzt. Nach Auswahl einer bestehenden Feature Geometrie wird automatisch ein semantisches Objekt erstellt, welches annotiert werden kann, vgl. "Neues semantisches Objekt erstellen". Wichtig hierbei ist, dass eine beschreibende Geometrie (das Feature), wie auch das prefLabel nicht optional ist.

Semantic Object Relation erstellen

Eine semantische Relation zwischen zwei semantischen Objekten wird ebenfalls durch eine Annotation beschrieben. Diese ist jedoch body-los. Das gewählte Prädikat der Beschreibung der Relation wird in beide Richtungen gleich verwendet. Somit ist eine vereinfachte Abfrage möglich.
