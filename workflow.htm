Vorbereitung

  • RDF Dateien erstellen
    • annotator agents ALS foaf:Agent | Beispiel
    • properties ALS rdf:Property | Beispiel
    • labels ALS skos:Concept (z.B. mit dem Labeling System) | Beispiel
    • expert data ALS rdfs:Resource (z.B. mit XTriples) | Beispiel
  • Upload der Dateien in die TripleStores (labeling.i3mainz.de - adwserv58.adwudlit.uni-mainz.de)
  • Änderung der JavaScript-Variablen RDF_agents, RDF_labels, RDF_resources, RDF_predicates in der index.htm
  • Erstellung der Features mit dem IBR Generic Viewer

Workflow

  • Bereits existierendes Feature annotieren
    • Annotator auswählen → Annotator
    • Spatialcontext und IBR Feature auswählen → Feature Object
    • preference Label (prefLabel) in einer Sprache des zu annotierenden semantischen Objekts angeben → Labeling
    • optional: altLabel, scopeNote des semantischen Objekts → Labeling, sowie Links zu Labeln, Expert Data, URIs oder Literale hinzufügen → Labeling & linking to Resources
    • Feature Annotation erzeugen → Graph wird erzeugt
  • Bereits existierendes semantisches Objekt annotieren
    • Annotator auswählen → Annotator
    • Semantic Object auswählen → Semantic Object
    • optional: prefLabel, altLabel, scopeNote des semantischen Objekts → Labeling, sowie Links zu Labeln, Expert Data, URIs oder Literale hinzufügen → Labeling & linking to Resources oder Feature hinzufügen → Feature
    • Semantic Object Annotation erzeugen → Graph wird erzeugt
  • Neues semantisches Objekt erzeugen
    • Annotator auswählen → Annotator
    • preference Label (prefLabel) in einer Sprache des zu annotierenden semantischen Objekts angeben → Labeling
    • optional: altLabel, scopeNote des semantischen Objekts → Labeling, sowie Links zu Labeln, Expert Data, URIs oder Literale hinzufügen → Labeling & linking to Resources
    • Semantic Object erzeugen → Graph wird erzeugt
  • Semantic Object Relation erstellen
    • Annotator auswählen → Annotator
    • 2x Semantic Object und Prädikat auswählen → Semantic Object
    • Relation erzeugen → Graph wird erzeugt
  • Annotation löschen
    • Annotation auswählen → Annotation
    • Annotationsgraph inkl. Metadaten und blank Node löschen → Graph wird gelöscht
